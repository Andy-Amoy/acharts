## Usage

  1. 引入cdn上的代码 
  ```js
    <script src="http://g.tbcdn.cn/bui/acharts/1.0.2/acharts-min.js"></script>
  ```

  2. 使用,引入acharts文件后，会在window上增加Chart变量代表Chart控件类

  ```js
  var chart = new Chart({
    //configs
  });
  // use acharts
  ```
